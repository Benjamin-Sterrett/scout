# Scout Research

## How to use

1. Copy the prompt from `discovery-prompt.md`
2. Run it in a separate Claude/Gemini/ChatGPT session (or multiple for cross-referencing)
3. Save raw output here as `findings-{source}-{date}.md`
4. Bring it back to a Scout session: "synthesize my discovery findings"

## Files

- `discovery-prompt.md` — The research prompt to run externally
- `findings-*.md` — Raw research output (add as you go)
- `synthesis.md` — Synthesized findings (generated after research)
